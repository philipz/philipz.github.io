About

Philipz（鄭淳尹），開源文化擁抱者，於 2013 年共同成立 Docker.Taipei，在接觸過 Docker 後，認為所有資訊相關人員都必須具備 Docker 操作技巧，遂發起 Docker Gitbook 共筆，尤其開源策略已成為軟體產業宣傳及行銷的手段，因此任何開源程式都可用 Docker 包裝，而官方的 Docker Hub 網站上有各式各樣的 Docker 映像檔可供下載使用，之後甚至還有微軟 Windows 平台的映像檔。從此，不必再煩惱安裝設定等枝微末節的「Dirty Jobs」，縮短IT人員的學習時間，直接切入核心軟體。

而 Docker 這三年的變化，尤其是 Machine 和 Swarm 工具，宣告正式進入分散式叢集時代，迅速就可啟動一整群的雲端服務，並以“精兵政策”來開發複雜系統了，專注於核心業務，不用像以前需要如此龐大的團隊，才能設計實作並維運整個分散式系統。

立即在網頁上體驗使用 Docker ^_^
